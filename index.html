CAPTURE THE FLAG
The Classic Field Game
RULES
  • Run into the enemy flag to pick it up, then carry it back to your base to score
  • Once you cross the centre line, enemies can tag you
  • Tagged players are frozen — a teammate must touch them to unfreeze
  • If a flag carrier is tagged, the flag drops and waits for an attacker to retrieve it
  • Win a round by capturing ALL enemy flags OR freezing ALL enemy players
CONTROLS
1 PLAYER
  • WASD — Move
  • F — Tag enemy / Unfreeze teammate
  • C — Switch player
  • ESC — Pause / Resume
2 PLAYERS — P2
  • IJKL — Move
  • ; — Tag enemy / Unfreeze teammate
  • . — Switch player
  • ESC — Pause / Resume
